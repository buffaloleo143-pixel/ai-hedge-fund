## 第四章：技术面五策略加权框架

### 4.1 五策略加权体系

技术分析Agent采用五策略加权集成体系，每条策略独立计算信号（bullish/neutral/bearish）与置信度（0-1），最终通过加权合成统一信号。

| 策略 | 权重 | 核心指标 | 判定逻辑 |
|------|------|----------|----------|
| 趋势跟踪 | **25%** | EMA(8)/EMA(21)/EMA(55) + ADX(14) | 短期与中期EMA同向 + ADX确认趋势强度 |
| 均值回归 | **20%** | Z-score(50日) + 布林带位置 + RSI(14/28) | Z<-2且BB<0.2→BULLISH；Z>2且BB>0.8→BEARISH |
| 动量 | **25%** | 1月/3月/6月动量 + 成交量动量 | 加权动量>5%且量增→BULLISH；<-5%且量增→BEARISH |
| 波动率 | **15%** | 历史波动率 + 体制比 + ATR比率 | 低波动体制(Z<-1)→BULLISH；高波动体制(Z>1)→BEARISH |
| 统计套利 | **15%** | Hurst指数 + 偏度 + 峰度 | H<0.4且skew>1→BULLISH；H<0.4且skew<-1→BEARISH |

### 4.2 信号合成逻辑

合成函数 `weighted_signal_combination` 将信号映射为数值（bullish=+1, neutral=0, bearish=-1），计算加权总分：

```
weighted_sum = Σ(信号值 × 权重 × 置信度)
final_score = weighted_sum / Σ(权重 × 置信度)
```

判定阈值：`final_score > 0.2 → BULLISH`，`< -0.2 → BEARISH`，否则 `NEUTRAL`。

#### 第六层：Portfolio Manager 的代码决策路径

Portfolio Manager 的代码逻辑（[portfolio_manager.py](file:///e:/qa/ai-hedge-fund/src/agents/portfolio_manager.py)）规定了以下决策路径：

1. **信号聚合**（L56-74）：将所有非 Risk Manager 的 Agent 信号压缩为 `{sig, conf, pred}` 格式
2. **价格共识计算**（L241-260）：对预测字段取均值，生成 `price_consensus`
3. **允许动作约束**（L107-168）：由 `compute_allowed_actions()` 确定性计算可执行操作及最大数量
4. **LLM 决策**（L263-290）：在约束范围内，由 LLM 综合信号、价格共识和允许动作做出最终判断

在本案中，决策链的关键节点：

- **信号层面**：6:6:6 完全均衡，LLM 无法从数量上获得方向性倾向
- **价格共识层面**："No price consensus for guidance"，均值参考价值丧失
- **允许动作层面**：HOLD 始终是合法选项（`actions["hold"] = 0`）
- **LLM 综合**：在信号均衡 + 价格共识失效的双重约束下，LLM 选择最保守的 HOLD 决策

这符合代码设计哲学：**当信号无法提供方向性指引时，默认持有是最理性的选择**。

### 7.3 为什么是 HOLD 而非 SHORT

这是一个比纯 API 版本更值得讨论的问题。在纯 API 版本中，9:3 的看空优势使 SHORT 成为自然而然的决策。但在 6:6:6 的均衡格局下，HOLD 而非 SHORT 的选择体现了 Portfolio Manager 的风险偏好：

**支持 HOLD 而非 SHORT 的理由**：

1. **信号均衡 = 不确定性高**：6:6:6 意味着系统对方向没有共识，在此条件下做空或做多都是对不确定性的单边押注
2. **看多信号质量提升**：Peter Lynch（85%）、Stanley Druckenmiller（78.5%）等高置信度看多信号的出现，使得"看多阵营只是噪音"的论点不再成立
3. **盈利增长的实际改善**：102.65% 的盈利增长和利润率扩张是真实发生的，不是数据错误，需要给予尊重
4. **HOLD 保留了灵活性**：在信息不充分时持有，等待更明确的信号（如下一季度财报确认趋势），优于在均衡信号下做出激进决策

**支持 SHORT 仍然存在的理由**：

1. **估值的刚性约束**：DCF 缺口 -75.8% 是客观事实，不会因为盈利改善而消失
2. **看多信号的可持续性存疑**：102.65% 的盈利增长大概率不可持续，一旦增速回归均值，GARP 框架将迅速转空
3. **价格共识偏空**：修正后的短期/中期/长期均值均低于当前价格

**Portfolio Manager 的最终权衡**：在信号完全均衡、价格共识失效的条件下，"不做比做好"成为最优策略。65% 的置信度反映了"倾向于不做但尚有不确定性"的判断。
